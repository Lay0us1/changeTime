某杀软会对时间较新且未扫描过的文件设置很高的可疑分数，通常大家只会改创建时间和修改时间，而virustotal还会识别**编译时间**，使用本脚本可一键修改**exe**、**dll**的**编译时间**、**创建时间**、**修改时间**和**访问时间**。

依赖：```pip3 install pypiwin32```
